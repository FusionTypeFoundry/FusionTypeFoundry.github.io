Fusion Type Foundry

Fusion Type is an independent type foundry where practical fonts and experimental ideas collide. The catalog jumps from clean, everyday typefaces built for text and branding to wild display fonts and odd little experiments that push typography into new territory. Every release is drawn and refined by hand, with an eye for both usability and personality. Some fonts want to disappear into the background, others demand all the attention, but together they show how type can be both a reliable tool and a playground for design. Whether it’s a serif, sans serif, or variable font, Fusion Type is about making typography that works hard, looks sharp, and has a little fun along the way.
